# Менеджер завдань

## Проблема
Багато людей забувають про свої завдання або не можуть ефективно їх організувати.

## Пітч
Цей застосунок допомагає швидко створювати, переглядати та відмічати виконані завдання в зручному інтерфейсі.

## MVP
Користувач може:
- додати нове завдання
- побачити список завдань
- відмітити завдання як виконане

## Стейкхолдери
Основним стейкхолдером є користувач, який використовує застосунок для створення та управління завданнями. Його основна потреба — швидко додавати, переглядати та відмічати виконані завдання.

Розробник відповідає за створення та підтримку продукту. Він зацікавлений у простоті реалізації та стабільній роботі.

Викладач оцінює якість виконання роботи, відповідність вимогам та логіку інтерфейсу.

Потенційні користувачі очікують простий, зрозумілий і зручний інтерфейс.

## Мета
Створити простий та зручний інтерфейс для роботи із завданнями.

## Критерії успіху
- Користувач може додати завдання без помилок
- Список завдань відображається коректно
- Інтерфейс зрозумілий без пояснень# -
